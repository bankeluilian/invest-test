# 金融投资性格测试

通过 25 道专业心理测试题，深度解析你的投资性格基因，匹配最适合你的投资大师风格。

## 功能特点

- **25 道专业测试题** - 基于投资心理学设计
- **4 种性格类型** - 价值投资型、风控防御型、系统均衡型、量化科技型
- **15 位投资大师** - 巴菲特、芒格、索罗斯、利弗莫尔等
- **个性化建议** - 根据测试结果给出专业投资建议

## 性格类型

| 类型 | 描述 | 匹配大师 |
|------|------|----------|
| 价值投资型 | 逆向投资，注重安全边际 | 巴菲特 |
| 风控防御型 | 风险厌恶，严格止损 | 利弗莫尔 |
| 系统均衡型 | 资产配置，纪律性强 | 都铎·琼斯 |
| 量化科技型 | 数据驱动，模型量化 | 格林布拉特 |

## 投资大师

涵盖巴菲特、芒格、格雷厄姆、费雪、林奇、卡拉曼、格林布拉特、索罗斯、德鲁肯米勒、利弗莫尔、保尔森、伯里、阿克曼、伊坎等 15 位传奇投资者。

## 使用方法

直接在浏览器中打开 `index.html` 即可开始测试。
